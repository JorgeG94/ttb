## As Voigt

**Description:** Converts a `Tensor2` or `Tensor4` data type to a reduced Voigt-storage `Tensor2s` or `Tensor4s`.

Possible calls:
- asvoigt(T)
- voigt(T)
- symstore(T) (due to backward compatibility)
